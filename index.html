How to get your URL — quick steps
  1. Open Google Sheets → create a blank sheet named "Legal Contract Tracker"
  2. Click Extensions → Apps Script
  3. Delete default code, paste everything from google-apps-script.js
  4. Click Deploy → New deployment → Web app
  5. Set "Who has access" to Anyone → click Deploy → copy the URL
  6. Paste the URL above and click Connect
